# Emmoloks Personal Portfolio 🌐

Welcome to my personal developer portfolio — built with [Hexo](https://hexo.io), customized and deployed using GitHub Pages. This site showcases my work, skills, and journey in tech as a software engineer, cybersecurity enthusiast, and cloud computing student.

🔗 **Live Site**: [https://emmoloks.github.io](https://emmoloks.github.io)

---

## 🚀 Features

- ✨ Minimal and responsive portfolio theme
- 🧠 Blog support for sharing knowledge & projects
- 💻 Projects section with real-world applications
- 📬 Contact form and social links
- ⛅ Deployed using GitHub Pages

---

## 🛠️ Tech Stack

- **Static Site Generator**: [Hexo](https://hexo.io)
- **Deployment**: GitHub Pages
- **Theme**: Custom-modified Hexo theme
- **Languages**: HTML, CSS, JavaScript, Markdown

---

## 📁 Folder Structure

```bash
.
├── source/             # All your markdown content and blog posts
├── themes/             # Custom or third-party theme used
├── _config.yml         # Global Hexo configuration
├── package.json        # Project dependencies
└── scaffolds/          # Content templates

```

## ⚙️ How to Run Locally
```
npm install -g hexo-cli
```
## Clone the Repository

git clone https://github.com/Emmoloks/emmoloks.github.io.git
cd emmoloks.github.io


## Install Dependencies
npm install

## Run the Site Locally
hexo serve

Then open http://localhost:4000 in your browser.
